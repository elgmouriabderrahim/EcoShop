# EcoShop API Documentation

Base URL: `http://localhost:8000/api`

Authentication uses Bearer tokens generated by Laravel Sanctum.

## Authentication

### Register
- `POST /auth/register`
- Body:
```json
{
  "full_name": "Alice Green",
  "email": "alice@example.com",
  "password": "password123",
  "password_confirmation": "password123"
}
```

### Login
- `POST /auth/login`
- Body:
```json
{
  "email": "alice@example.com",
  "password": "password123"
}
```

### Logout (auth)
- `POST /auth/logout`

### Profile (auth)
- `GET /auth/me`

## Public Catalog

### List categories
- `GET /categories`

### List products
- `GET /products?category_id=1&per_page=15`

### Product details
- `GET /products/{id}`

## Cart (auth)

### View cart
- `GET /cart`

### Add/update item in cart
- `POST /cart`
- Body:
```json
{
  "product_id": 1,
  "quantity": 2
}
```

### Update quantity
- `PATCH /cart/{cartId}`
- Body:
```json
{
  "quantity": 3
}
```

### Remove item
- `DELETE /cart/{cartId}`

## Orders (auth)

### Place order from current cart
- `POST /orders`
- Triggers:
  - `OrderPlaced` event
  - queued `SendOrderConfirmationEmailJob`
  - queued `UpdateOrderStockJob`

### User order history
- `GET /orders`

## Admin Endpoints (auth + admin)

### Products
- `GET /admin/products`
- `POST /admin/products`
- `GET /admin/products/{id}`
- `PUT/PATCH /admin/products/{id}`
- `DELETE /admin/products/{id}`

### Categories
- `GET /admin/categories`
- `POST /admin/categories`
- `GET /admin/categories/{id}`
- `PUT/PATCH /admin/categories/{id}`
- `DELETE /admin/categories/{id}`

### Orders
- `GET /admin/orders`
- `GET /admin/orders/{id}`

## Status Codes
- `200` success
- `201` created
- `401` unauthenticated
- `403` forbidden
- `422` validation/business rule errors

## Default Seeded Users
- Admin: `admin@ecoshop.test` / `password`
- Customer: `customer@ecoshop.test` / `password`
